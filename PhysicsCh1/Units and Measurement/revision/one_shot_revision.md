## Quick Review Summary

### 7 Fundamental SI Units
1. **Length** → meter (m)
2. **Mass** → kilogram (kg)
3. **Time** → second (s)
4. **Temperature** → kelvin (K)
5. **Electric Current** → ampere (A)
6. **Luminous Intensity** → candela (cd)
7. **Amount of Substance** → mole (mol)

## Key Formulas to Remember

### Least Count
- Vernier: LC = 1 mm / VSD
- Screw gauge: LC = Pitch / Circular divisions

### Error Analysis
- Addition/Subtraction: Δz = Δx + Δy
- Multiplication/Division: Δz/z = Δx/x + Δy/y
- Powers: Δz/z = n × (Δx/x)

### Significant Figures
- Multiplication → fewest SF
- Addition → fewest decimal places

## Key Concepts

1. **Dimensional Analysis:** Check if equation is dimensionally consistent
2. **Error Propagation:** How uncertainties combine in calculations
3. **Significant Figures:** How many digits to write in answer
4. **Zero Error:** Instrument calibration correction

---

## JEE Quick Points

- Always check dimensions before calculating
- Remember: relative errors add in multiplication/division
- Absolute errors add in addition/subtraction
- Small angle approximations: sin θ ≈ θ (in radians)
- Dimensions of trigonometric functions are 0 (they're ratios)
