## 1. Trigonometry for Physics

### 1.1 Basic Trigonometric Ratios

| Ratio | Definition | Formula |
|-------|------------|---------|
| sin θ | Opposite/Hypotenuse | sin θ = perpendicular/hypotenuse |
| cos θ | Adjacent/Hypotenuse | cos θ = base/hypotenuse |
| tan θ | Opposite/Adjacent | tan θ = sin θ/cos θ |
| cosec θ | 1/sin θ | Hypotenuse/Opposite |
| sec θ | 1/cos θ | Hypotenuse/Base |
| cot θ | 1/tan θ | Base/Perpendicular |

### 1.2 Important Identities

> [!IMPORTANT]
> **Pythagorean Identity:**
$$\sin^2\theta + \cos^2\theta = 1$$

**Other Identities:**
- $\tan\theta = \frac{\sin\theta}{\cos\theta}$
- $\cot\theta = \frac{\cos\theta}{\sin\theta}$
- $\sec^2\theta = 1 + \tan^2\theta$
- $\cosec^2\theta = 1 + \cot^2\theta$

### 1.3 Angle Formulas

| Formula | Application |
|---------|--------------|
| $\sin(A+B) = \sin A \cos B + \cos A \sin B$ | Adding angles |
| $\cos(A+B) = \cos A \cos B - \sin A \sin B$ | Adding angles |
| $\sin 2\theta = 2\sin\theta\cos\theta$ | Double angles |
| $\cos 2\theta = \cos^2\theta - \sin^2\theta$ | Double angles |

### 1.4 Small Angle Approximations

For small θ (in radians):
- $\sin\theta ≈ \theta$
- $\cos\theta ≈ 1 - \frac{\theta^2}{2}$
- $\tan\theta ≈ \theta$

## 2. Logarithms and Exponents

### 2.1 Logarithm Rules

| Rule | Formula |
|------|---------|
| Product | $\log(ab) = \log a + \log b$ |
| Quotient | $\log(a/b) = \log a - \log b$ |
| Power | $\log(a^n) = n \log a$ |
| Change of base | $\log_b a = \frac{\log a}{\log b}$ |

### 2.2 Natural Logarithm

- $\ln x$ means $\log_e x$
- $e = 2.71828...$
- $\ln(e^x) = x$
- $e^{\ln x} = x$

### 2.3 Exponential Decay/Growth

> [!KEY-CONCEPT]
> Many physical phenomena follow exponential laws:
> - Radioactive decay: $N = N_0 e^{-\lambda t}$
> - Capacitor discharge: $V = V_0 e^{-t/RC}$
> - Damped oscillations: $x = A e^{-\gamma t} \cos(\omega t)$

## 3. Calculus Basics for Physics

### 3.1 Derivatives

| Function | Derivative |
|----------|-------------|
| $x^n$ | $nx^{n-1}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $e^x$ | $e^x$ |
| $\ln x$ | $1/x$ |
| Constant | 0 |

### 3.2 Integration

| Function | Integral |
|----------|----------|
| $x^n$ | $x^{n+1}/(n+1) + C$ |
| $1/x$ | $\ln x + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $e^x$ | $e^x + C$ |

### 3.3 Physics Applications

- **Derivative:** Velocity = dx/dt, Acceleration = dv/dt
- **Integration:** Displacement = ∫v dt, Work = ∫F dx

## 4. Vectors Basics

### 4.1 Vector Operations

| Operation | Formula |
|-----------|---------|
| Addition | $\vec{A} + \vec{B} = \vec{C}$ |
| Subtraction | $\vec{A} - \vec{B} = \vec{D}$ |
| Magnitude | $|\vec{A}| = \sqrt{A_x^2 + A_y^2}$ |
| Dot Product | $\vec{A} \cdot \vec{B} = AB\cos\theta$ |
| Cross Product | $|\vec{A} \times \vec{B}| = AB\sin\theta$ |

### 4.2 Component Form

$$\vec{A} = A_x \hat{i} + A_y \hat{j} + A_z \hat{k}$$
$$A_x = A\cos\alpha, A_y = A\cos\beta, A_z = A\cos\gamma$$

## 6. JEE Important Points

### 6.1 Key Formulas to Remember

1. **Trigonometry:** $\sin^2\theta + \cos^2\theta = 1$
2. **Logarithms:** $\log(a^b) = b \log a$
3. **Derivatives:** $\frac{d}{dx}(x^n) = nx^{n-1}$
4. **Integrals:** $\int x^n dx = \frac{x^{n+1}}{n+1}$

### 6.2 Common Mistakes

> [!COMMON-MISTAKE]
> ❌ **Mistake:** Using degrees instead of radians in calculus
> ✅ **Correct:** Convert to radians for differentiation/integration
> **Why:** Calculus formulas derived using radians

> [!COMMON-MISTAKE]
> ❌ **Mistake:** Confusing vector magnitude with components
> ✅ **Correct:** Use Pythagorean theorem for magnitude
> **Why:** Components are projections, not full values
