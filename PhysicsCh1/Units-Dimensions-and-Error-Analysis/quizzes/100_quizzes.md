## Quiz 10: True/False
**Questions:** 10
**Time:** 5 minutes

1. [T/F] All physical quantities have dimensions. (Answer: False - angle is dimensionless)
2. [T/F] Significant figures indicate accuracy. (Answer: False - indicate precision)
3. [T/F] Errors always add in subtraction. (Answer: True)
4. [T/F] Dimensional analysis can find the value of π. (Answer: False)
5. [T/F] Random errors can be eliminated by taking more readings. (Answer: True - reduced, not eliminated)
6. [T/F] Zero error is a systematic error. (Answer: True)
7. [T/F] 1 kg = 1000 g is exact. (Answer: True - by definition)
8. [T/F] LC of vernier = 1/n where n = VSD divisions. (Answer: True)
9. [T/F] 27°C = 300 K exactly. (Answer: False - 300 K = 27°C, but 300 K = 26.85°C exactly)
10. [T/F] Force and energy have same dimensions. (Answer: False - different)
