## Error Analysis Formulas

### Error Types

| Error Type | Description | Formula |
|-----------|-------------|---------|
| Absolute error | Magnitude of deviation | $\Delta a = |a - a_0|$ |
| Mean absolute error | Average of absolute deviations | $\overline{\Delta a} = \frac{\sum |\Delta a_i|}{n}$ |
| Random error (mean) | Half the range | $\Delta a_{mean} = \pm \frac{a_{max} - a_{min}}{2}$ |
| Relative error | Error relative to measured value | $\delta a = \frac{\Delta a}{a}$ |
| Percentage error | Relative error in percent | $\delta a \times 100\%$ |

### Error Combination Formulas

| Operation | Result | Error Formula |
|-----------|--------|--------------|
| $Z = A + B$ | $Z = \bar{A} + \bar{B}$ | $\Delta Z = \Delta A + \Delta B$ |
| $Z = A - B$ | $Z = \bar{A} - \bar{B}$ | $\Delta Z = \Delta A + \Delta B$ |
| $Z = A \times B$ | $Z = \bar{A} \times \bar{B}$ | $\frac{\Delta Z}{Z} = \frac{\Delta A}{A} + \frac{\Delta B}{B}$ |
| $Z = A / B$ | $Z = \bar{A} / \bar{B}$ | $\frac{\Delta Z}{Z} = \frac{\Delta A}{A} + \frac{\Delta B}{B}$ |
| $Z = A^n$ | $Z = (\bar{A})^n$ | $\frac{\Delta Z}{Z} = |n| \cdot \frac{\Delta A}{A}$ |
| $Z = A^{1/n}$ | $Z = \sqrt[n]{\bar{A}}$ | $\frac{\Delta Z}{Z} = \frac{1}{|n|} \cdot \frac{\Delta A}{A}$ |
| $Z = A^m \cdot B^n$ | — | $\frac{\Delta Z}{Z} = |m| \cdot \frac{\Delta A}{A} + |n| \cdot \frac{\Delta B}{B}$ |

## Measuring Instruments Formulas

### Vernier Calipers

| Formula | Description |
|---------|-------------|
| $1 \text{ MSD} = 1 \text{ mm}$ | Main Scale Division |

## Significant Figures Formulas

| Rule | Formula |
|------|---------|
| Addition / Subtraction | Round to **least decimal place** |
| Multiplication / Division | Round to **least significant figures** |
| Exact numbers | Infinite significant figures |
| Scientific notation | $N \times 10^n$, where $1 \le N < 10$ |

**Standard rounding (when digit < 5 vs ≥ 5):**
- $< 5$: Drop, keep preceding unchanged
- $\geq 5$: Round up
- $= 5$ (followed by zeros): Round to nearest even
