# One-Shot Revision - Units, Dimensions and Measurement

## Ultra-Condensed Notes

### 1. Systems of Units
- **SI Base Units (7):** kg, m, s, A, K, mol, cd
- **CGS:** cm, g, s → Force: dyne (10⁻⁵ N), Energy: erg (10⁻⁷ J)
- **Conversions:** km/h ÷ 3.6 = m/s; g/cm³ × 1000 = kg/m³; °C + 273 = K
- **Prefixes:** k (10³), M (10⁶), G (10⁹), T (10¹²), m (10⁻³), μ (10⁻⁶), n (10⁻⁹)

---

### 2. Dimensional Analysis
- **Fundamental Dimensions:** M, L, T, I, K, N, C
- **Key Formulas:**
  - Force: MLT⁻²
  - Energy/Work: ML²T⁻²
  - Power: ML²T⁻³
  - Pressure: ML⁻¹T⁻²
  - Momentum: MLT⁻¹
  - Density: ML⁻³
  - Acceleration: LT⁻²
- **Principle:** All terms in equation must have same dimensions
- **Cannot find:** Value of dimensionless constants (π, e, 2)

---

### 3. Significant Figures
- **Rules:**
  - Non-zero always significant
  - Zeros between non-zero significant
  - Leading never significant
  - Trailing after decimal significant
  - Trailing without decimal ambiguous
- **Operations:**
  - Addition/Subtraction → least decimal places
  - Multiplication/Division → least SF
- **Rounding:** Round to nearest even when exactly 5

---

### 4. Errors in Measurement
- **Types:** Random (reduce by averaging), Systematic (identify and correct)
- **Formulas:**
  - Error in sum: Δ(A+B) = ΔA + ΔB
  - Error in difference: Δ(A-B) = ΔA + ΔB
  - Error in product: ΔZ/Z = ΔA/A + ΔB/B
  - Error in power: ΔZ/Z = n × ΔA/A
- **Relative Error:** Δx/x
- **Percentage Error:** (Δx/x) × 100%

---

### 5. Measuring Instruments
- **Vernier LC:** 1 MSD / n (standard: 0.1 mm)
- **Screw Gauge LC:** Pitch / divisions (standard: 0.01 mm with 0.5 mm pitch, 100 divisions)
- **Zero Error:** Positive → subtract, Negative → add

---

## Must-Remember Formulas

| Quantity | Formula |
|----------|---------|
| Vernier LC | 1 MSD / n |
| Screw Gauge LC | Pitch / divisions |
| Error sum | ΔZ = ΔA + ΔB |
| Error product (relative) | ΔZ/Z = ΔA/A + ΔB/B |
| Error power | ΔZ/Z = n × ΔA/A |
| km/h to m/s | ÷ 3.6 |
| g/cm³ to kg/m³ | × 1000 |
| °C to K | + 273 |

---

## Last-Minute Points

1. **Dimensional Analysis**: Check dimensions FIRST to eliminate wrong options
2. **Errors**: Remember - errors ALWAYS ADD in both sum AND difference
3. **Significant Figures**: Use correct rule for operation type
4. **Zero Error**: Check BEFORE taking any measurement
5. **Conversions**: Use 3.6 for speed, 1000 for density, 273 for temperature

---

## Memory Triggers

- **"SUMDIFF = ADD"**: Sum or Difference, errors always ADD
- **"LSSSF"**: Least Significant Significant Figures for multiplication
- **"LARS"**: Least Absolute Rounding by Scale (decimal places) for addition
- **"Positive = Subtract"**: Positive zero error - subtract to correct
- **"3.6 for speed"**: km/h to m/s divide by 3.6
