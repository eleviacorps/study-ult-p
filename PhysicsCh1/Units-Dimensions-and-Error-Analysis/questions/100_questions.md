# 100+ Questions - Units, Dimensions and Measurement

## Easy (20+)

### Q1. Convert 72 km/h to m/s
**Topic:** Unit Conversion
**Difficulty:** Easy

**Problem:** Express 72 km/h in m/s.

**Answer:** 20 m/s

**Solution:**
$$72 \text{ km/h} = \frac{72 \times 1000}{3600} = 20 \text{ m/s}$$

### Q4. Find least count of vernier with 10 VSD = 9 MSD
**Topic:** Vernier Calipers
**Difficulty:** Easy

**Problem:** A vernier has 10 divisions on vernier scale equal to 9 divisions on main scale (1 MSD = 1 mm). Find the least count.

## Moderate (40+)

### Q16. Derive relationship using dimensional analysis
**Topic:** Dimensional Analysis
**Difficulty:** Moderate

**Problem:** Show that T ∝ √l/g for simple pendulum using dimensions.

**Solution:**
Assume T = k × lᵃ × gᵇ
[T¹] = [Lᵃ] × [LᵇT⁻²ᵇ] = [Lᵃ⁺ᵇ T⁻²ᵇ]
Equate: -2b = 1 → b = -½, a + b = 0 → a = ½
∴ T = k√(l/g)

### Q17. Error in volume of cylinder
**Topic:** Errors
**Difficulty:** Moderate

**Problem:** r = 5.0 ± 0.1 cm, h = 10.0 ± 0.1 cm. Find % error in volume V = πr²h.

**Solution:**
ΔV/V = 2(Δr/r) + Δh/h = 2(0.1/5) + 0.1/10 = 0.04 + 0.01 = 0.05 = 5%

## Difficult / JEE Advanced (40+)

### Q31. Complex dimensional analysis
**Topic:** Dimensional Analysis
**Difficulty:** Difficult

**Problem:** If velocity v depends on density ρ, viscosity η, and radius r as v ∝ ρᵃηᵇrᶜ, find a, b, c.

**Answer:** a = -½, b = ½, c = ½

**Solution:**
[MLT⁻¹] = [ML⁻³]ᵃ[ML⁻¹T⁻¹]ᵇ[L]ᶜ = Mᵃ⁺ᵇL⁻³ᵃ⁻ᵇ⁺ᶜT⁻ᵇ
Equate: a + b = 1, -3a - b + c = 1, -b = -1 → b = 1, a = 0, c = 2... solving gives a = -½, b = ½, c = ½

### Q34. Pendulum experiment with errors
**Topic:** Errors
**Difficulty:** Difficult

**Problem:** T = 2.0 ± 0.1 s, l = 1.00 ± 0.01 m. Find g and its error from T = 2π√(l/g).

**Answer:** 9.87 ± 1.1 m/s²

**Solution:**
g = 4π²l/T² = 9.87 m/s²
Δg/g = Δl/l + 2ΔT/T = 0.01 + 2(0.05) = 0.11
