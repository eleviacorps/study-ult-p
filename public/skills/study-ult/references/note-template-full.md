## Section-by-Section Guidelines

### Why This Topic? (20-30 lines)
- Start with BIG QUESTION
- Historical context (who, when, why)
- What problem this solves
- KEY-CONCEPT callout

### Definition (30-40 lines)
- Full mathematical definition
- Table of symbols
- Physical interpretation
- Units explanation

### Derivation (40-50 lines)
- Step-by-step from first principles
- Code block format for clarity
- Each step explained

### Properties (40-50 lines)
- 4+ properties with detail
- >[!DEEP-INSIGHT] callout

### Cases (40-50 lines)
- 3+ common cases
- Formula + when to use

### Examples (150+ lines) — 3 EXAMPLES REQUIRED
Each example needs:
- Full problem statement
- Given/Find
- Approach (how to think)
- Step-by-step solution
- >[!{EXAM_NAME}-INSIGHT] callout
- Answer with units

### Quick Rules (20-30 lines)
- 3+ rules
- >[!TIP] callout for memory aids

### Comparisons (20-30 lines)
- Table format
- 4+ aspects compared

### Common Mistakes (60-80 lines) — 5+ MISTAKES REQUIRED
Each mistake needs:
- Wrong approach
- Correct approach
- Why wrong
- How to avoid
- >[!COMMON-MISTAKE] callout

### {EXAM_NAME} Patterns (40-50 lines)
- 4+ patterns
- What to look for
- Approach for each

### Formula Summary (20-30 lines)
- Table with all formulas
- Key points column

### Total: 400+ lines minimum
